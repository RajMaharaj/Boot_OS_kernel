Boot os kernel

This repository contains the codes of booting a linux kernel from scratch in C and asm.

Welcome to Boot OS kernel Project.

This project deals with making an Operating System from scratch using basic concepts of C, ASM, x-86 architecture, calling convention for x86, qemu (x86 emulator), gdb (debugging), ctags (Navigating code) and taking help from http://www.linuxfromscratch.org/. This is a very basic project and to just experiment with some new concepts and learnings as well to understand the working of operating system specially when it starts its boot process.
